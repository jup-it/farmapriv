# farmapriv
Se crea una aplicacion web en jsf que permita la implementacion de una farmacia con los siguientes modulos:

1.- Inventario
2.- Compras *
   2.1.- Proveedores
   2.2.- Ordenes de Compra
   2.3.- Reportes
3.- Tienda online
   3.1.- Pedidos
   3.2.- Promociones
   3.3.- Clientes
   3.4.- Administracion de Tienda
   3.5.- Reportes
4.- Entrega a domicilio
   4.1.- Rutas de entrega
   4.2.- Reportes
